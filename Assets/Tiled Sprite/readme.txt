Tiled Sprite
============

1. Drag "Tiled Sprite" prefab to your scene.

2. Set size.

3. Set tile picture (that may or may not be packed in an atlas).

4. Set preferred material.

5. Optional. Change other settings as needed.

You can also change any setting at run time. Please, refer to the provided example for more details.

A custom clipping shader is provided for convenience. Use it as you like. Give credits if you share it.

In case of any issues or questions, don't be afraid to write to: support@equinoxcreations.com
